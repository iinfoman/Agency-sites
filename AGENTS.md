# Agency-sites — Agent Instructions

This is the main workspace for building client and agency websites.

## Workspace structure
- Each project has its own folder inside `Agency-sites/`
- Current projects: `Assistant-H/`
- Skills are in `.cursor/skills/` (Cursor-active) and mirrored in `ProjectName/Skills/`

## How I work
- Always read the relevant skill file before building a site
- Build HTML sites using the `html-site` skill standards
- Commit and push to GitHub after completing any meaningful work
- Ask for brand colors and site purpose before starting any new site

## GitHub
- Remote: https://github.com/iinfoman/Agency-sites
- Default branch: `master`
- Push after every completed feature

## Preferences
- Vanilla HTML/CSS/JS first — no frameworks unless asked
- Mobile-first, accessible, fast
- Clean file names (lowercase, hyphens)
- No unnecessary comments in code
